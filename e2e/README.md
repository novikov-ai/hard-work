# E2E ваше все

[Добавил](https://github.com/novikov-ai/seasonal-vector-framework/commit/f66b8d30f754bf2995500a4530b491b7f834db63) в проект e2e-тесты на основе [Playwright](https://playwright.dev/).

## Сценарии

1. Онбординг — приложение показывает стартовый экран при
первом запуске, загружает пример данных и запоминает, что
пользователь уже прошёл онбординг

2. Выполнение задач — галочка на действии запускает анимацию
затухания и кнопку Undo, отмена возвращает действие обратно,
а через 5 секунд оно исчезает

3. Создание плана — добавление домена, вектора и действия на
карте сразу отображается во вкладке Today

## Эффект

- Каждый пуш в main автоматически прогоняет эти тесты перед
деплоем — если что-то сломалось, деплой не пройдёт
- Можно смело рефакторить код и сразу видеть, не сломал ли ты
основные сценарии
- Локально — npm run test:e2e даёт быструю проверку перед
пушем
